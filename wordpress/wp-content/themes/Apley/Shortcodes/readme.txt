This file explains how to get the widgets used in this template:

/*  1. TWITTER  */

DOWNLOAD
---------------

http://wordpress.org/extend/plugins/twitter-widget-pro/

Author: Ran.ge
Author URI: http://ran.ge/
Plugin URI: http://wordpress.org/extend/plugins/twitter-widget-pro/

INSTALL:
---------------

* Upload the entire Twitter-widget-pro.zip file to the /wp-content/plugins/ directory, you can do this by going to plugins -> add new.

* Activate the plugin through the 'Plugins' menu in WordPress.

* You will find a 'twitter' widget in your WordPress 'Widgets' menu.



/*  2. CONTACT FORM 7  */

DOWNLOAD
---------------

http://wordpress.org/extend/plugins/contact-form-7/

Author: Takayukister
Author URI: http://ideasilo.wordpress.com/
Plugin URI: http://wordpress.org/extend/plugins/contact-form-7/

How to INSTALL:
---------------
* Upload the entire contact-form-7.zip file to the /wp-content/plugins/ directory, you can do this by going to plugins -> add new.

* Activate the plugin through the 'Plugins' menu in WordPress.

* You will find 'Contact' menu in your WordPress admin panel.


/*  3. SHORTCODES */

How to INSTALL:
---------------
* Upload the entire PlayneShortcodes.zip file to the /wp-content/plugins/ directory, you can do this by going to plugins -> add new.

* Activate the plugin through the 'Plugins' menu in WordPress.

* You can now insert the plugins in your wordpress text editor as described in the main read me file.
